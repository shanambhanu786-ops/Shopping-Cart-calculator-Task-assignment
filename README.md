# 🛒 Shopping Cart Calculator

A responsive Shopping Cart Calculator built using **HTML, CSS, and JavaScript**. This project demonstrates DOM manipulation, form validation, loops, event handling, and dynamic price calculation.

---

## 📌 Features

- Responsive shopping cart interface
- Product listing with prices
- Quantity input validation
- Automatic item total calculation
- Dynamic grand total update
- Checkout confirmation
- Prevents invalid quantity input
- Maximum quantity validation
- Modern UI design
- Beginner-friendly project structure

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 Functionalities

- Displays multiple products.
- Calculates item totals automatically.
- Updates grand total instantly.
- Validates quantity input.
- Checkout confirmation popup.
- Handles invalid user input.

---

## ✔ Validations

- Quantity cannot be less than 1.
- Maximum quantity allowed is 100.
- Invalid values are automatically corrected.
- Empty inputs are reset to valid values.
